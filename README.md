
# Logic Gates Simulator

### Features

- All basic logic gates
- A save/load system
- Cross platform
- Truth table generator
- Equation generator
- Boolean algebra simplifier

### Planned Features

- Grid snap system
- Minimap for large circuits
- Keyboard shortcuts
- Component quick access
- Undo/Redo functionality
